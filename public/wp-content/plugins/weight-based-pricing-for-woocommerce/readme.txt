A simple weight based pricing plugin for WooCommerce. Set different prices for different weight ranges easily

== Description ==

Weight based pricing plugin for WooCommerce helps you define your inventory in weight and set prices based on weight ranges. 

== KEY FEATURES ==

* Simple weight based pricing rules
* Multiple weight ranges and weight units
* Minimum and maximum weight options
* Set intervals / steps for the customers to choose
* Support for wastage percentage 

== Getting started ==

* Install and activate the plugin
* Go to your WordPress dashboard -> WooCommerce -> Weight based pricing & configure the options
* Create or edit a product to start configuring the weight based pricing rules

== Installation ==

Just use the WordPress installer or upload to the /wp-content/plugins folder. Then Activate the Weight based Pricing for WooCommerce plugin.

= Minimum Requirements =

* WordPress 4.9.0 or greater
* WooCommerce 3.0.0 or greater
* PHP version 5.6.0 or greater
* MySQL version 5.0 or greater

== Frequently asked questions ==

= Does it support variable product type? =

Yes. You can set weight based pricing for each variant
